一、水平居中

1.行盒、行块盒的水平居中：text-align:center

span文字 a元素

2、常规流块盒的水平居中：定宽然后左右margin设置auto

3、绝对定位元素的水平居中：定宽然后left=right=0,margin:auto

4、浮动元素，只能调整左右外边距;不好

二、垂直居中

1.单行文本垂直居中:设置行高等于所在区域的高度

单行文本垂直居中

2、行盒内多行文本垂直居中：做不到

3、块盒、行块盒多行文本垂直居中：设置上下padding相同

块盒或者行块盒内的多行文本垂直居中，只能设置上下padding来做； 但是这样做的缺点是，没法知道这和盒子的具体高度。

4、绝对定位的垂直居中:top=bottom=0,定高
