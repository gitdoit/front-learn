3D旋转

层叠上下文：

按照我目前的理解，perspective设置在父元素上创建层叠上下文，也可以理解为3d舞台。
一个立方体放在舞台中心，你只能看到一个面，x轴左右移动的时候你能看到两个面。所以啊
你把一个正方形放在舞台左边，在按照x轴旋转45度，会发现他是向右后方倒下去的，而不是直接倒下去的。
这是因为他在舞台的左边啊，你坐在观众席的中心位置。懂我意思吗？

  • 2d旋转只能按照某一点进行正逆时针，3d旋转可以按照3个轴进行旋转
  • 沿着x轴进行旋转rotateX(180deg)，向里为正，向外为负
    2D旋转
    3D旋转
  • 沿着y轴进行旋转rotateY(180deg),逆时针为正
    3D旋转
  • 沿着z轴进行旋转rotateZ(180deg)，和2d旋转一样按照默认的transform-origin来转，就是中心点；
    可以通过改变这个原点来改变z轴位置
    3D旋转
  • 矢量旋转:rotate3D(x,y,z,deg)
    3D旋转

3D旋转之3D呈现

子元素为3D效果，旋转父元素的时候默认会把子元素二维化，通过设置呈现效果，保持子元素3D形态
