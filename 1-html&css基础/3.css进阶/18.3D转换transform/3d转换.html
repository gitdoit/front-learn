很重要的一点啊：默认屏幕的中心点就是你面对的平面。想象一个正方体，正面面对你的时候你只能看到前面
那么此时这个正方体就在原点上，如果这个正方体靠屏幕最左侧，那么你能看到他的右面和前面两个面了。

  • 原点(transform-origin)默认在元素中心点，即(50%*width,50%height)处
  • x为水平轴向右为正，y为垂直轴向下为正，z为垂直屏幕轴向外为正。
  • 透视两要素：
    • 人眼到屏幕的距离(perspective),越近值越小，z轴上的变化就越明显。单纯的改变
    • 物体在z轴上移动的距离，值越大物体越大
  • perspective属性需要定义在被透视元素的父元素上
