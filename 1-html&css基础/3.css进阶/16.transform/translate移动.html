translate函数介绍

  • transform Vs 相对定位

    transform配合translate用来做元素的移动，虽然相对定位也能达到相同的效果(移动时不影像其他元素)
    但是后者是用来做布局使用的，前者在元素移动画面渲染方面有更好的性能，且移动的最小单位可以超过相对定位的1px限制，以
    更平滑的方式来移动。
  • translate不能用于行内元素
  • translate的百分比取值相对于自身的宽/高；translateX(50%)向右平移自身宽度的一半。

transform: translate(110px,44px);
transform: translate(50%,0);
width: 400px; transform: translateX(50%);
