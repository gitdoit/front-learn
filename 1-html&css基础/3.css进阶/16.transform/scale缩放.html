scale的优势

我们可以通过变化宽高来达到同样的目的，但是同样，这还是会影响到周围的元素。

并且，宽高的变化无法控制 变化原点 ，变化时它的顶部会固定不动

transform可以同时接收位移、旋转、缩放函数，但是要将位移函数放在首位

transform: scale(2)

以中心点开始，向四周变大为当前宽高的两倍

transform: scale(1,2)

以中心点开始,宽不变，高变为2倍

变化原点

默认以中心点为变化原点，向四周扩散，同样可以通过 transform-origin:left bottom 来控制
