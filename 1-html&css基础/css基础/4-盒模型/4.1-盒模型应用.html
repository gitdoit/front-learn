一、改变宽高范围

根据设计图量元素宽高的时候，量的是整个元素的宽高，但是设置height\width的时候是 内容宽高，这样再加上内边距和边框宽高之后就和设计图对不上了；

解决办法1：为宽高手动加减边框和内边距占据的大小占

解决办法2：CSS3的box-sizing:boeder-box;

英雄联盟

二、改变背景颜色覆盖范围

content

  • background-clip: padding-border;(默认)背景只覆盖边框盒
  • background-clip: content-box;背景只覆盖内容盒
  • background-clip: padding-box;背景只覆盖填充盒

三、溢出处理

宽高限定死，内容又超长，溢出后默认会显示出来；可以通过overflow:xxx;来规定溢出后的处理方式

  • hidding：溢出隐藏
  • scroll:使用滚动条（有没有溢出都有滚动条）
  • auto:自动，没溢出的时候不出现滚动条，溢出后才出现
