定位：手动控制元素再包含块中的精准位置

position属性

一、static:默认值，静态定位即不定位（不会脱离文档流）

二、relative:相对定位（不会脱离文档流）

相对定位只是对应自己的位置进行偏移;
即相对于自己原来的位置向上下左右四个方向进行偏移；
偏移只是视觉上的效果，并不会把其他元素挤开； 如果偏移冲突了，那么以上、左为准
通常用来为绝对定位做嫁衣

相对定位左偏移100px、上偏移50px
无定位，margin-left : 100 px

三、absolut:绝对定位(脱离文档流)

1、宽高为auto时，适应内容；

2、绝对定位的元素的包含块是它从它爹开始往上找，第一个碰到的定位元素(postition不是static的);
如果找不到，那么整个网页就是他的初始包含块。

四、fixed:固定定位

固定定位和绝对定位除了包含块不同其他完全相同；
固定定位的包含块是浏览器的可视窗口，就是你眼睛能够看得见的那部分网页，例如
你按下f12，可视窗口就被控制台挤小了。但是网页大小没变

固定定位元素

五、固定、绝对定位下的居中

总结：绝对定位和固定定位中，当上下或左右为0时，margin为auto时会自动吸收掉剩余的空间。
和常规流的width对于margin的关系类似。

1、定宽高
2、水平居中-将左右距离设置为0，margin为auto

3、垂直居中-将上下距离设置为0，margin为auto

4、水平+垂直居中-上下左右距离设置为0，margin为auto

一个脱离了文档流的元素具有如下特性：

1、文档流中的元素摆放时，会自动忽略脱离的文档流的元素。

2、文档流中的元素计算自动高度时，会忽略脱离了文档流的元素。 这两点和浮动元素一样，浮动元素也是脱离了文档流的元素
