包含块

每个元素都有它的包含块，包含块决定了盒子的排列区域，绝大部分情况下盒子的包含块为其 父元素的内容盒

规则一、每个块盒的总宽度，必须等于包含块的宽度

1、如果没指定块盒的宽度，那么默认值是auto，意味着如果外边距、边框、内边距没有用光包含块的宽度，那么剩下的就分给width

2、如果块盒指定了宽度，并且如果再加上外边距、边框、内边距都没分完包含块的宽度，那么剩余的将分配给 margin-right

3、如果指定了块盒宽度，并设置 margin-right:auto;margin-left:auto; 那么剩余的宽度平均分给左右外边距

规则二、块盒垂直方向上的auto值

1、对于height来说，时应内容高度

2、对于margin来说，就等于0

规则三、百分比取值

1、对于padding\margin\的四个方向及width的百分比； 意味着按照 包含块的宽度 计算(和高度无关);

2、对于高度height的百分比
1)、如果包含块高度自适应，那么子元素高度百分比无效
2)、如果包含块高度指定了，此时子元素高度百分比按照包含块的高度计算

margin-top:50% margin-left:50%

规则四、上下外边距合并

两个常规流块盒，如果是挨在一起中间没有任何东西隔开的话(包括Border)，那么他们的外边距 会进行合并，即使一个在另一个内部。

margin-top:100px 没有效果，因为和父元素的margin-top合并了。
margin-top:100px 有效果，因为碰到了红块的content
