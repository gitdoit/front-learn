四、背景图

有时候使用背景图片和img元素好像都可以，但是需要按照一下两个原则选择

1、图片属于网页内容时，使用img，例如广告、文章中的插图
2、图片仅用于美化页面时，使用背景图。例如按钮元素的背景图

背景图涉及到下面几个css属性

1、background-img: url('');设置背景图片
2、background-clip: border-box; 背景覆盖范围
3、background-size: content/cover/100px 100px/50% 50%;背景图片大小
4、background-position: left top/center bottom/4px 3px;背景图片位置
5、background-attachment: 控制背景图是否固定；
6、background: url() no-repeat center/100% fixed #000,路径 是否重复，位置，尺寸，固定，背景色。顺序固定，但不是每个属性都要写，不写的可以省略。如果位置和尺寸都有，必须先写位置

使用雪碧图，及很多小图片合成大图片，然后利用属性只展示指定位置就行
